# RxDatePicker
安卓自定义DatePicker日期选择器
# 框架引入
### 先在 build.gradle(Project:XXXX) 的 repositories 添加:
~~~
repositories {
    maven {
        url "https://jitpack.io"
    }
}
~~~
### 然后在 build.gradle(Module:app) 的 dependencies 添加: 
~~~
dependencies {
	implementation 'com.github.404NotFuond:RxDatePicker:1.0.0'
}
~~~
